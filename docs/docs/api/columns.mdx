# Columns

To define columns, either pass an array of column definitions or a helper function.
The latter is useful if you are using TypeScript because it infers the column's value's type from the first argument.

For example:

```tsx
<Table
  items={items}
  id="id"
  columns={[
    {
      id: 'firstName',
      header: 'First name',
      value: (item) => item.firstName,
    },

    {
      id: 'lastName',
      header: 'Last name',
      value: (item) => item.lastName,
      renderCell: (lastName: string) => <b>{lastName}</b>,
    },
  ]}
/>
```

or

```tsx
<Table
  items={items}
  id="id"
  columns={(col) => [
    col((item) => item.firstName, {
      id: 'firstName',
      header: 'First name',
    }),

    col((item) => item.lastName, {
      id: 'lastName',
      header: 'Last name',
      renderCell: (lastName /* type string is inferred */) => <b>{lastName}</b>,
    }),
  ]}
/>
```
